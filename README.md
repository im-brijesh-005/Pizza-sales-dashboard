Pizza Sales Report Dashboard

Project Overview

The Pizza Sales Report Dashboard is an analytical and visualization project designed to evaluate pizza sales performance using historical transaction data. The dashboard presents key business metrics, trends, and category-level insights to support data-driven decision-making.
This project focuses on converting raw sales data into a structured, interactive report that highlights performance patterns across time, product categories, and pizza sizes.
________________________________________
Objectives
•	Analyze overall revenue and order performance

•	Identify high-performing pizza categories and sizes

•	Examine weekly and monthly order trends

•	Provide clear, actionable insights through KPIs and visualizations
________________________________________
Key Performance Indicators (KPIs)

•	Total Revenue: 817.86K

•	Total Orders: 21K

•	Total Pizzas Sold: 50K

•	Average Order Value: 38.31

•	Average Pizzas per Order: 2.32

Analytical Insights

Sales Trends

•	Weekly analysis indicates higher order volumes toward the end of the week

•	Monthly trends reveal seasonal fluctuations in demand

Category Performance

•	The Classic category contributes the highest share of total sales

•	Supreme and Veggie categories demonstrate stable performance

•	Chicken category records comparatively lower sales volume
Size Distribution

•	Small and Medium pizzas account for the majority of sales

•	XL and XXL sizes show minimal contribution
